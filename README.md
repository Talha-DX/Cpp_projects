🚀 How to Run the Projects
1. **Clone the repository**
   ```bash
   git clone https://github.com/Talha-DX/Cpp_projects.git
   ```
2. **Navigate to the project folder**
   ```bash
   cd Cpp_projects
   ```
3. **Compile and run any project**
   ```bash
   g++ project_name.cpp -o output_name
   ./output_name
   ```
